Bug Fix Test Cases

Test 1: Queenside Castling

  1. Start a new game
  2. Play these moves: e4, e5, Nf3, Nc6, Bc4, Bc5, d3, d6, Bd2, Bd7, Nc3, Nf6, Qe2, Qe7
  3. Click on the white King (e1)
  4. Click on c1 (two squares to the left)
  5. ✅ The king should castle queenside (King on c1, Rook on d1)

Test 2: Undo Selection Bug

  1. Start a new game
  2. Click on e2 pawn (should highlight)
  3. Click on e4 (pawn moves)
  4. Click "Undo" button
  5. ✅ The board should NOT have any piece selected (no yellow outline)
  6. ✅ You should be able to select any piece now

Test 3: Pawn Promotion

  1. Use "Force Move (SAN)" to set up a position near promotion
  2. Enter these moves one by one: e4, d5, exd5, Nf6, d6, e6, dxe7, Qxe7, d4, Qe4+, Be2, Qxg2, Bf3, Qxh1+, Kd2, Qxf3, Nf3, h5, d5, h4, d6, h3, d7, h2, d8=Q
  3. Now move the h2 pawn to h1
  4. ✅ A prompt should appear asking "Promote to (q/r/b/n):"
  5. ✅ Enter "q" and the pawn should become a queen
